# 数据集说明

本项目使用以下数据集（未包含在仓库中，请自行下载）：

| 数据集 | 说明 | 下载地址 |
|--------|------|----------|
| E-commerce_dataset | 电商对话意图分类数据集 | https://github.com/SophonPlus/ChineseNlpCorpus |
| JDDC | 京东多轮对话语料 | https://jddc.jd.com/ |
| online_shopping_10_cats | 电商评论情感分析（10类） | https://github.com/SophonPlus/ChineseNlpCorpus |

## 使用方式

下载后将文件放置在项目根目录下即可。
